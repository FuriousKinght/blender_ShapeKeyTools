## 插件简介

形态键工具（ShapeKeyTools）是一款专为MMD（MikuMikuDance）工作流设计的Blender插件，提供强大的形态键（Shape Key）参数控制和关键帧设置功能。支持智能多选操作，让动画师能够高效地批量处理形态键动画。

## 功能特性

### 🎯 核心功能

- **智能多选系统**
  - 支持 Shift+点击 连续选择多个形态键
  - 支持 Ctrl+点击 切换单个形态键的选择状态
  - 多选状态下支持批量操作

- **批量操作**
  - 批量设置多个形态键的值
  - 批量插入关键帧
  - 批量删除关键帧
  - 一键清除所有选择

- **形态键搜索与过滤**
  - 按名称搜索形态键
  - 按类别过滤（嘴、眉、目、材质变形等）
  - 显示中文翻译名称

- **形态键监测面板**
  - 自动监测当前时间线范围内使用的形态键
  - 显示形态键当前值和状态
  - 支持手动刷新和一键重置

### 🎬 关键帧预设模式

插件提供9种动画预设模式，简化形态键动画制作：

| 模式     | 说明                                   |
| -------- | -------------------------------------- |
| 简单振荡 | 基础振荡模式，在最小值和最大值之间切换 |
| 正弦波   | 平滑的周期性变化                       |
| 方波     | 开关式变化                             |
| 三角波   | 线性渐变                               |
| 锯齿波   | 单向渐变                               |
| 随机变化 | 随机值变化，支持平滑度调节             |
| 缓入缓出 | 平滑加速减速                           |
| 弹跳     | 模拟弹跳效果                           |
| 呼吸     | 模拟呼吸节奏                           |

### 🔧 其他功能

- 跳过分界线形态键（以 `---` 开头的名称）
- 实时界面重绘
- 选择锚点记忆
- 多选数值统一控制滑块

## 安装步骤

### 方法一：从ZIP文件安装

1. 下载插件的ZIP压缩包
2. 打开 Blender
3. 进入 `编辑 > 首选项 > 插件`
4. 点击右上角的 `安装` 按钮
5. 选择下载的ZIP文件
6. 勾选启用插件

### 方法二：手动安装

1. 解压插件文件夹
2. 将 `ShapeKeyTools` 文件夹复制到 Blender 的插件目录：
   - Windows: `C:\Users\你的用户名\AppData\Roaming\Blender Foundation\Blender\4.5\scripts\addons\`
   - macOS: `~/Library/Application Support/Blender/4.5/scripts/addons/`
   - Linux: `~/.config/blender/4.5/scripts/addons/`
3. 打开 Blender，进入 `编辑 > 首选项 > 插件`
4. 搜索 "形态键工具" 并启用

## 使用指南

### 面板位置

插件面板位于：**视图3D > 属性面板 > MMD > 形态键工具**

### 基本操作

1. **选择形态键**
   - 单击：选择单个形态键
   - Shift+点击：连续选择多个形态键
   - Ctrl+点击：切换选择状态

2. **批量设置值**
   - 选择多个形态键后，使用多选数值滑块统一设置值

3. **生成动画**
   - 在关键帧设置区域选择目标形态键
   - 设置起始帧和结束帧
   - 选择预设模式
   - 点击"生成形态键动画"

4. **使用监测面板**
   - 监测面板显示当前时间线范围内使用的形态键
   - 可以快速查看和重置形态键值

## 已知问题与限制

1. **分界线形态键**：以 `---` 开头的形态键会被自动跳过
2. **参考键**：基础形态键（reference_key）不会被操作
3. **动画数据**：需要对象具有形态键和动画数据才能使用关键帧功能
4. **搜索限制**：搜索功能仅匹配形态键名称或翻译名称
